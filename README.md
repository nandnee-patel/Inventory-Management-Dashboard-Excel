# 📊 Inventory Management Dashboard (Excel Data Analytics Project)

## Project Overview
This project presents an interactive **Inventory Management Dashboard** built in Microsoft Excel to analyze inventory and stock performance across multiple dimensions such as product, shelf, and time period.

The goal of this project is to transform raw inventory data into actionable insights using **PivotTables, Power Pivot, and dashboard visualization techniques**, without relying on Excel formulas.

---

## Business Problem
Organizations often face challenges in managing inventory effectively, such as:

- Monitoring stock levels across multiple products and locations  
- Identifying products that are understocked or overstocked  
- Understanding stock movement trends over time  
- Supporting operational decisions with real-time inventory insights  

This dashboard consolidates inventory data into an interactive view, enabling quick analysis and decision-making.

---

## Dataset
The dataset contains inventory transactional data including:

- Product Name / ID  
- Shelf / Storage Location  
- Quantity Received  
- Quantity Dispatched  
- Quantity Available  
- Stock Movement Dates  

The data was analyzed and aggregated using Excel's **PivotTables and Power Pivot**, allowing dynamic reporting and visualization.

---

## Tools & Techniques Used
**Microsoft Excel**  

Key techniques applied:

- PivotTables for dynamic summaries  
- Power Pivot for data modeling and connecting multiple tables  
- Pivot Charts for visualization  
- KPI Metrics and Inventory Analysis  
- Interactive Filters / Slicers for product and shelf selection  
- Dashboard design and layout for executive-ready presentation  

---

## Dashboard Features
The dashboard provides:

- **Inventory performance overview**  
- **Stock level monitoring** for products and shelves  
- **Product-wise and shelf-wise analysis**  
- **Interactive filtering** for dynamic exploration of data  
- **Trend analysis** to visualize stock movement over time  

These features allow supply chain professionals to quickly explore patterns, monitor stock health, and make data-driven decisions.

---

## Key Insights
Example insights derived from the dashboard:

- Certain products consistently face low stock levels, indicating restocking needs  
- Some shelves are overstocked, revealing inventory allocation inefficiencies  
- Stock movement trends highlight seasonal demand fluctuations  
- Inventory monitoring can prevent stockouts and optimize warehouse space  

These insights can help organizations improve **inventory planning, replenishment decisions, and operational efficiency**.

## Business Value
This project demonstrates the ability to:

- Transform raw inventory data into meaningful insights  
- Build interactive, professional dashboards in Excel  
- Communicate inventory performance metrics visually  
- Support data-driven decision making in supply chain management
